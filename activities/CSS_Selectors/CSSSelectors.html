Lyrics

1. The CSS selection methods are as follows:

  • By element
  • By ID
  • By class
  • y element.class
  • By all (*)

2. Class names can be used by more than one HTML element, but IDs can only be used on one.

3. Span tags are used in-line, and will not result in a line break. Div tags are for discrete blocks of text or content, and will result in a line break.

4. Header, Main and Footer tags differ from Div tags in name only. They perform exactly the same function, but have different names to improve legibility.

5. Colors can be selected by color name, by RGB values or by Hex values.
