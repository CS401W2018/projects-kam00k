Behold, a Rabbit.

1. .jpeg files are good for pictures that requre greater color depth than .gif files, but do not need to be of high quality.

2. .gif files are good for storing simple images or animations in a small space, but support only 256 colors.

3. .png files support an alpha channel, which allows them to have transparent areas.

4. Images can be resized in the .css file using a class or other selector.

5. Resizing a large photo in CSS will not shrink the file itself, but resizing in editing software will. Images can be large, and the less data the client has to download, the better.
