Relative Links

This is a relative link.

Relative links are "relative" to the location of the current webpage.
This is one of the reasons they can use rules such as . and ..

The 3 rules of relative links are as follows:

  1. . = Current directory (can be used, but redundant)
  2. .. = Parent directory (Above the current one)
  3. / = Start at root directory
