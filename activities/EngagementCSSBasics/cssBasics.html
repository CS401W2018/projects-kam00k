Answers to engagement questions

  1. Inline CSS is declared within an individual set of paragraph tags. Embedded CSS is declared within the head tags at the top of the file. External CSS is pulled from a separate file altogether.
  2. Only external CSS should be used, because the others have been deprecated and are no longer considered best practice.
  3. Selectors are used to select the html elements whose styles the dev wants to change.
